Managing High Run Load

Heavy run weeks: Drop main lift sets from 4 to 3. Skip plyometrics if legs feel dead. Always keep the core and injury-prevention work — low fatigue, high value.

After hard runs: 48 hours minimum between hard efforts. Schedule gym on easy run days.

Progression

Add 2.5–5 kg to main lifts weekly while form holds. For plyometrics, improve height and landing quality before adding box height. Deload every 6–8 weeks (50% volume for one week).

★ Key Additions for Runners

Nordic Hamstring Curl — Reduces hamstring strains by ~50%. The eccentric loading is the whole point.

Tibialis Raise — Shin splint insurance. Vastly underused.

Eccentric Calf Raise — Achilles resilience under load. The slow lowering is the stimulus.

Jump Squats & Box Jumps — Plyometrics improve running economy 3–8%. Do them fresh.

Hip Abduction — Prevents IT band syndrome and knee pain from hip drop.

Copenhagen Plank — Adductor/groin strength. Non-negotiable for high-mileage athletes.
